# Deep Research Agent

Deep Research Agent is a fork of LangChain's Open Deep Research project, extended into a more complete research assistant product.

The project keeps the original LangGraph-based deep research workflow and builds on it with broader model support, OpenCode Go compatibility, safer report generation, and plans for a dedicated local-first user interface.

## Goals

- Run deep research across multiple model providers.
- Support additional low-cost and open model providers.
- Save research outputs, notes, sources, and reports locally.
- Let users configure keys and model settings from the app UI.
- Allow users to continue chatting with the agent after a final report is generated.

## Current Status

The core research agent is working through LangGraph Studio. The next major step is building a custom UI for running research, managing settings, saving reports, and chatting with completed research.
